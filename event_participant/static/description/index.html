Participants administration for event

Adds participants for event registration

This module is a extention of event and adds participants administraion to Odoo standard Events. The module makes a connection between event and partner (participant) using the registration. The standard behaivor for Event is that each registration can be made by a administator of the customer on behaf for several attendees.

With this module it is possible to add individual participants on each registration. This means that each event has a list of participants who attend to it. Keeping track of individuals makes it possible to create attendants lists, to register what cources and what level of education an individual has attended to and achieved. Each event registration has a list of participants who are belong to the company which has registrated to the event. You can print participant lists for cource leader and lists to hand out to attendees.

When the event is ended, system will give each participant a tag that shows which event he / she has attended. The tags is named from the event type.

Sister modules

With the module event_participant_diploma, diplomas can be printed for each participant. The template for the diploma are made in the publishing system Scribus that gives you high quality originals for printing.

With the module event_participant_labels, you can print name tags for the participants to hand out. You can use almost any type of name tags sold in office stores. The template for the name tag are made in the program Glabels and its easy to change brand of sheet labels and configure the information you want to be printed on the tag.

The modules event_participant_labels and event_participant_diploma depends on the modules report_glabels and report_scribus.
