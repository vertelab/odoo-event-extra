Participants list for event registration

This module is a extention of event. Makes a connection between event and partner (participant). Each event has a list of participants who attend to it.

Each event registration has a list of participants who are belong to the company which has registrated to the event. When the event is end. System till give each participant a tag that shows which event he / she has been.
